# TicTacToe with Epitech

## Install

update `pip` with `python3`
```bash
python3 -m pip install --upgrade pip
```

Install `pygame`
```bash
python3 -m pip install -U pygame --user
```

Clone repositori
```bash
git clone https://github.com/grcpils/TicTacToe.git tictactoe
cd tictactoe
```

Execute the game
```bash
./TicTacToe.py
```
